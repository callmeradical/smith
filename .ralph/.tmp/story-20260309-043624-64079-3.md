### US-003: Add common developer tooling to image
Status: in_progress
Depends on: US-001

Description:
As an engineer running loops, I want common development tools preinstalled so scripts and diagnostics work consistently across loops.

Acceptance Criteria:
- [ ] Install required packages in Dockerfile with explicit commands, including `apk add --no-cache bash git curl jq ca-certificates make python3 py3-pip ripgrep nodejs npm` (or Alpine-equivalent package names available in target repo).
- [ ] Install pnpm support via approved mechanism (corepack enable or explicit package install) and document which method was used.
- [ ] Example: running tool checks in container returns success for git, curl, jq, make, node, npm/pnpm, python3, pip, and rg.
- [ ] Negative case: if any required tool command is missing, smoke test reports the exact missing binary and fails.
- [ ] Keep final image layer changes minimal by cleaning package caches where applicable.
